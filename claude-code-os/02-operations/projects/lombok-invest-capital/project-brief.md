**Client:** Lombok Invest Capital
**Start Date:** December 2024
**Status:** Active
**Primary Contact:** Benito

---

## Project Overview

**Goal:** Build MSG Scraper to monitor investment communications and alert on critical information

**Scope:**
- Scrape Microsoft Outlook MSG files for investment-related communications
- Extract key information from supplier messages
- Email notifications for critical keywords
- Dashboard for viewing scraped data

**Success Criteria:**
- Real-time alerts on critical supplier communications
- Accurate data extraction from MSG files
- User-friendly interface for daily monitoring
- Production-ready by mid-December 2024
